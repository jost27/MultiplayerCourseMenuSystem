## Auto Login
### access the editor settings by clicking on the three vertical dots near the play button and selecting “Advanced Settings…”
![image](https://github.com/user-attachments/assets/cf4e3179-4e19-4f87-9b68-3bb9968044d8)
### Now scroll down to Level Editor → Play. Locate the Additional Launch Parameters and enter -AUTH_TYPE=”accountportal”.
![image](https://github.com/user-attachments/assets/fe0df861-ec7b-4bae-a3c5-9ef159bb59ff)
### Now launch the game in standalone mode. When the game starts, you should see the overlay open asking you to sign in:
![image](https://github.com/user-attachments/assets/7ea083a4-8127-4e3e-abfa-fa6ddec7a1d2)
